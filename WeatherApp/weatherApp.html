Why SkyCast? ☀️

SkyCast is a clean, modern Android weather app that uses your current location to deliver real-time forecasts. Whether you're searching for weather in a different city or checking your local temperature, it's fast, accurate, and beautifully built using Jetpack Compose and modern Android architecture.

📍

Location Weather

Auto-fetch current location and show weather instantly using GPS.

🔍

City Search

Search any city to see its current temperature, humidity, and wind speed.

🧱

Clean Architecture

Built with MVVM, repositories, use-cases, and separation of concerns.

🛠️

Retrofit API

Fetches real-time weather data through Retrofit and OpenWeatherMap API.

💉

Dependency Injection

Integrated with Dagger Hilt for DI, ViewModel and repository injection.

🌙

Dark Theme

Elegant UI with full dark mode support using Jetpack Compose theming.

👨‍💻 Developer Notes

From the Developer

SkyCast was created as a modern, scalable weather app using all recommended Android best practices. I wanted to build a clean, efficient UI using Jetpack Compose and back it with a solid MVVM + DI structure.

It’s the perfect template for building any API-based app in Kotlin with scalable architecture and testability.

🛠️ Built With

Kotlin Jetpack Compose MVVM Retrofit Dagger Hilt OpenWeatherMap API
