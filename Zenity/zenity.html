Why Zenity? 💬

Zenity is a mental health app designed for quick, safe, and verified therapist access. Built to be deployed in 4 days, Zenity focuses on the core essentials: connecting users with licensed professionals, booking therapy sessions, chatting securely, and engaging with wellness communities.

🔐

User Authentication

Signup/Login for patients and therapists with secure Firebase Authentication.

👩‍⚕️

Therapist Profiles

View verified therapist profiles with specialization and availability.

📅

Session Booking

Patients can request sessions with therapists.

💬

Realtime Chat

Secure 1-on-1 chat between patients and therapists using Firebase Realtime DB.

🌐

Community Forum

Public thread-based forum for wellness discussions and support.

🛡️

Admin Panel

Admins can review and verify therapists to ensure trust and safety.

👨‍💻 Developer Notes

🧘‍♂️

From the Developer

Zenity was built as a fast, focused MVP mental health app using Firebase and MVVM. Its goal is to prove how quickly essential care connections can be built and tested in the real world.

The admin panel allows us to ensure all therapists are verified before sessions begin, ensuring a safe platform for all users.

🛠️ Built With

Kotlin MVVM Firebase Auth Realtime DB Firestore SharedPreferences

🐙 Open Source

Zenity’s MVP is open to feedback and contributions. Therapists, developers, and users alike can help improve the platform for everyone.
