Why Quantum Chat? 🔐

Quantum Chat is a fast and modern real-time messaging app with secure authentication and private chat features. Connect with anyone using email and password login, and enjoy seamless, Firebase-powered conversations.

🔐

Secure Authentication

Sign up or log in using email & password. Firebase Auth keeps your data safe.

💬

Realtime Chat

Send and receive messages instantly using Firebase Realtime Database.

👥

Private Messaging

One-on-one private chats built with Fragments and RecyclerView.

⚡

Lightweight

Optimized for performance and minimal storage usage (just 8 MB).

📥 Download & Installation

System Requirements

  • 📱 Android 6.0 or higher
  • 💾 10 MB free storage
  • 🌐 Internet connection for messaging

Installation Instructions

  1. Download the APK file
  2. Go to Settings → Security → Unknown Sources
  3. Enable "Install unknown apps"
  4. Open the downloaded APK file
  5. Tap "Install" and start chatting! 📡
